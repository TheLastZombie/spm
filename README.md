# Usage
<!-- usage -->
```sh-session
$ npm install -g sps
$ sps COMMAND
running command...
$ sps (-v|--version|version)
sps/0.1.0 win32-x64 node-v12.18.3
$ sps --help [COMMAND]
USAGE
  $ sps COMMAND
...
```
